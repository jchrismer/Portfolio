I developed a video based US speed limit sign detection algorithm using OpenCV. The algorithm starts by using a Fast Radial Symmetric Transform (FRST) to find rectangles that have a similar aspect ratio to that of a speed limit sign. Detected rectangles are sent to a cascade classifier to quickly be rejected or identified as a speed limit sign. Objects that pass the classifier are finally fed into a Convolutional Neural Network to have their speed read and reported to the user. The embedded video shows a demonstration of the program in action.

Source code can be found on the project on github. A write up can be found at my project blog
