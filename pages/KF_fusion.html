The goal of this project was to fuse a 3-axis accelerometer, gyroscope and magnetometer into a quaternion describing the sensor's orientation. I successfully developed both an Extended Kalman Filter (EKF) and Unscented Kalman filter (UKF) for the purpose of sensor fusion. Unique issues using quaternion states in the UKF arose and were dealt with in a mathematically consistent way. The EKF employed an adapative scheme to compensate for large errors during heavy movements.

Source code can be found on the project on github. A write up can be found at my project blog
