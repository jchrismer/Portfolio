As part of another project, I needed an effictive calibration routine for a MEMS gyroscope, magnetometer and accelerometer. I implemented a routine which can calibrate an accelerometer and gyroscope for bias, scale factors and non-orthagonality. The routine is simple to use and does not require any tools to administer. The magnetometer has hard iron and soft iron distortions compensated using ellipsoid fitting. Lastly I also implemented a magnetometer bias calibration routine for automatic and real time hard iron compensation.

Source code can be fond on the project on github. A write up can be found at my project blog
